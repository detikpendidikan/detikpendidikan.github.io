10. cos (90⁰ + A) + sin (180⁰ + A) + tan (180⁰ + A) + cotan (90⁰ + A) samadengan....


Dit :
cos (90⁰ + A) + sin (180⁰ + A) + tan (180⁰ + A) + cotan (90⁰ + A)

Penyelesaian :
Ingat bahwa
sin, cos , tan , cot , sec, csc apabila
(90 +/- A) atau (270 +/- A) akan berubah identitas, dimana :
sin -> cos
cos -> sin
tan -> cot
cot -> tan
sec -> csc
csc -> sec

dan
sin, cos , tan , cot , sec, csc apabila
(180 +/- A) atau (360 +/- A) identitas tetap, dimana :
sin -> sin
cos -> cos
tan -> tan
cot -> cot
sec -> sec
csc -> csc

Namun perhatikan KUADRANNYA
Kuadran 1 -> Semua positif
Kuadran 2 -> hanya sin dan csc yang positif

kuadran 3 -> hanya tan dan cot

kuadran 4 -> hanya sec dan cos

maka
cos (90⁰ + A) + sin (180⁰ + A) + tan (180⁰ + A) + cotan (90⁰ + A)
= -sin A + (- sin A) + tan A + (- tan A)
= -sin A - sin A + tan A - tan A
= -2 sin A+ 0
= -2 sin A

26. 11. Diketahui cotan teta = -3/4 dengan 90° < teta < 180°. Nilai sec teta adalah


Kategori Soal : Matematika - Trigonometri
Kelas : X (1 SMA)
Pembahasan :
Pembagian sudut dalam trigonometri.
Dalam trigonometri, besar sudut α dibagi menjadi 4 kelompok, yaitu :
Kuadran I : 0° < α < 90° (0 < α < π/2)
sin (90° - α) = cos α
cos (90° - α) = sin α
tan (90° - α) = cotan α
cotan (90° - α) = tan α
secan (90° - α) = sec α
sec (90° - α) = cosec α

Kuadran II : 90° < α < 180° (π/2 < α < π)
sin (180° - α) = sin α
cos (180° - α) = -cos α
tan (180° - α) = -tan α
cotan (180° - α) = -cotan α
secan (180° - α) = secan α
sec (180° - α) = -sec α

Kuadran III : 180° < α < 270° (π < α < 3π/2)
sin (270° - α) = -cos α
cos (270° - α) = -sin α
tan (270° - α) = cotan α
cotan (270° - α) = tan α
secan (270° - α) = -sec α
sec (270° - α) = -cosec α

Kuadran IV : 270° < α < 360° (3π/2 < α < 2π)
sin (360° - α) =sin (-α) = -sin α
cos (360° - α) = cos (-α) = cos α
tan (360° - α) = tan (-α) = -tan α
cotan (360° - α) = cotan (-α) = -cotan α
secan (360° - α) = secan (-α) = -secan α
sec (360° - α) sec (-α) = sec α

Mari kita lihat soal berikut.

Perhatikan gambar terlampir.
cotan θ = -3/4, 90° < θ < 180° (kuadran II)

PQ² + QR² = PR²
⇔ 4² + 3² = PR²
⇔ PR² = 16 + 9
⇔ PR² = 25
⇔ PR = 5

sin θ = 4/5
cos θ = -3/5
cosec θ = 5/4
sec θ = -5/3
tan θ = -4/3

27. Bentuk sederhakan dari sin (90°-A) + cos180 (180°-A) - tan (270°+A) + cotan (90°-A) Tolong bantuin Kaka dan jangan ngasal ya Kaka baik :(​


Jawaban:

Kita dapat menggunakan identitas trigonometri dasar untuk menyederhanakan bentuk tersebut. Berikut adalah langkah-langkahnya:

1. sin(90°-A) = cos(A) --> Identitas sin-kos

2. cos(180°-A) = -cos(A) --> Identitas cos

3. tan(270°+A) = -cot(A) --> Identitas tan-cot

4. cot(90°-A) = tan(A) --> Identitas cot-tan

Maka, substitusi identitas-identitas tersebut ke dalam bentuk awal, didapatkan:

sin(90°-A) + cos(180°-A) - tan(270°+A) + cotan(90°-A)

= cos(A) + (-cos(A)) - (-cot(A)) + tan(A)

= cos(A) + cos(A) + cot(A) + tan(A)

= 2cos(A) + (cos(A)/sin(A)) + (sin(A)/cos(A))

= 2cos(A) + (cos^2(A)+sin^2(A))/(sin(A)cos(A))

= 2cos(A) + 1/(sin(A)cos(A))

= 2cos(A) + sec(A)cosec(A)

Maka, bentuk sederhana dari sin(90°-A) + cos(180°-A) - tan(270°+A) + cotan(90°-A) adalah 2cos(A) + sec(A)cosec(A)

semoga bermanfaat
