10. lam ta'rif berjumpa tsa'


Tajwid

Al-Qamariyah dan Al-Syamsiyah
__________________________

✏ Singkat saja, lam ta’rif bertemu huruf ث merupakan hukum bacaan “Al-Syamsiyah"

▪ Al-Qamariyah
Yaitu bila lam ta’rif (ال) bertemu dengan salah satu huruf
ا ب غ ح ج ك و خ ف ع ق ي م هـ
(disingkat: اَبْغِ حَجَّكَ وَخَفْ عَقِيْمَه)
Al-qamariyah disebut juga dengan idzhar qamariyah karena cara membacanya “jelas”
Contoh:
الْحَمْدُ ــ الْعَالَمِيْنَ ــ الْفَلَقِ

▪ Al-Syamsiyah
Yaitu bila lam ta'rif (ال) bertemu dengan salah satu huruf
ت ث د ذ ر ز س ش ص ض ط ظ ل ن
Al-syamsiyah disebut juga dengan Idgham syamsiyah
Contoh:
الرَّحْمٰنِ ــ وَالتِّيْنِ ــ وَالزَّيْتُوْنِ

21. Tolong dibantu kak.. Usm stis


∫dx/(e^x + 1)

misal, e^x + 1 = u
e^x = u - 1
e^x dx = du
dx = du/(e^x)
dx = du/(u - 1)

∫dx/(e^x + 1) = ∫(du/(u - 1))/u
= ∫du/(u(u - 1))
= ln|u - 1| - ln|u| + C
= ln |(u - 1)/u| + C
= ln|e^x/(e^x + 1)| + C
= ln|(e^x + 1)/(e^x)|^(-1) + C
= -ln|(e^x + 1)/(e^x)| + C
= -ln|1 + 1/e^x| + C
= -ln|1 + e^(-x)| + C

Jawaban tidak ada dalam opsi.

= 3 + 1/(18/11)

= 3 + 1/(1 + 1/(11/7))
= 3 + 1/(1 + 1/(1 4/7))
= 3 + 1/(1 + 1/(1 + 4/7))
= 3 + 1/(1 + 1/(1 + 1/(7/4)))
= 3 + 1/(x + 1/(y + 1/z))
x = 1, y = 1, z = 7/4
xyz = 1 . 1 . 7/4 = 7/4
