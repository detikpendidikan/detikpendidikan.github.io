8. tolong bantu kk, . latihan soal tiu cpns


Jawaban:

D. 80

Penjelasan dengan langkah-langkah:

jarak = 500 km

kecepatan = x km/jam

waktu = w + 3/4

jarak = kecepatan x waktu

500 = x (w + 3/4)

500 = xw + 3/4x ........ persamaan I

jarak = 400

kecepatan = x km/jam

w = w - 1/2

400 = x (w-1/2)

400 = xw -1/2x .............. persamaan II

persamaan I - Persamaan II (metode eliminasi xw)

100 = 3/4x - (-1/2x)

100 = 3/4x + 1/2x

100 = (3+2) /4 x

100 = 5/4 x

x = 100 x 4/5

x = 80 km/jam (D)

s1 = 500 km

w1 = t + 3/4 jam

s2 = 400 km

w2 = t - 1/2 jam

s1/w1 = s2/w2

500/(t + 3/4) = 400/(t - 1/2)

500(t - 1/2) = 400(t + 3/4)

500t - 250 = 400t + 300

500t - 400t = 300 + 250

100t = 550

t = 550/100

t = 5,5 jam

v1 = s1/w1

v1 = s1/(t + 3/4)

v1 = 500/(5,5 + 3/4)

v1 = 500/(5,5 + 0,75)

v1 = 500/6,25

v1 = 80

jadi x adalah 80 km/jam (D)

14. baris document root/var/www/html​


Baris document pada folder root/var/www/html​ adalah letak direktori dimana kita bisa menaruh halaman website kita. Folder root/var/www/html​ merupakan folder dimana halaman-halaman website diakses. Folder tersebut mirip dengan folder htdocs pada aplikasi web server lokal seperti XAMPP. Sistem operasi Linux memang identik dengan web server, beberapa distro linux yang sering digunakan untuk server website adalah

RedhatDebianUbuntuCentos

Untuk sistem operasi Windows sendiri juga memiliki sistem operasi yang dikhususkan untuk server, yaitu Windows Server.

Pembahasan

Wesbite merupakan salah satu layanan yang ada di internet. Selain website, layanan-layanan lainnya yang terdapat pada internet adalah

EmailMarketplaceMedia sosialMiling listE banking

Fasilitas-fasilitas di internet yang sudah disebutkan di atas merupakan layanan yang secara umum sudah banyak digunakan oleh banyak pengguna internet. Untuk menggunakan layanan-layanan di atas, kita memerlukan email untuk mendaftarnya.

Pelajari lebih lanjut

Demikian pembahasan mengenai apa itu baris pada document root/var/www/html​, baris dokumen tersebut adalah halaman sebuah website. Untuk mempelajari lebih lanjut mengenai web server lainnya dapat dibaca pada link berikut

39. jelaskan perbedaan antara WWW, HTML, dan HTTP


HTML (Hyper Text Markup Language) adalah sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web dan menampilkan berbagai informasi di dalam sebuah browser Internet. Bermula dari sebuah bahasa yang sebelumnya banyak digunakan di dunia penerbitan dan percetakan yang disebut dengan SGML (Standard Generalized Markup Language), HTML adalah sebuah standar yang digunakan secara luas untuk menampilkan halaman web. HTML saat ini merupakan standar Internet yang didefinisikan dan dikendalikan penggunaannya oleh World Wide Web Consortium (W3C).

HTML berupa kode-kode tag yang menginstruksikan browser untuk menghasilkan tampilan sesuai dengan yang diinginkan. Sebuah file yang merupakan file HTML dapat dibuka dengan menggunakan browser web seperti Mozilla Firefox atau Microsoft Internet Explorer. HTML juga dapat dikenali oleh aplikasi pembuka email ataupun dari PDA dan program lain yang memiliki kemampuan browser.


HTTP Singkatan dari Hypertext Transfer Protocol, yang mana adalah suatu protokol yang digunakan oleh World Wide Web. HTTP mendefinisikan bagaimana suatu pesan bisa diformat dan dikirimkan dari server ke client.

HTTP juga mengatur aksi-aksi apa saja yang harus dilakukan oleh web server dan juga web browser sebagai respon atas perintah-perintah yang ada pada protokol HTTP ini.
Sebagai contoh, ketika Anda mengetikkan suatu alamat atau URL pada internet browser Anda, maka sebenarnya web browser akan mengirimkan perintah HTTP ke web server.

Web server kemudian akan menerima perintah ini dan melakukan aktivitas sesuai dengan perintah yang diminta oleh web browser (misalnya akses ke database, file, e-mail dan lain sebagainya). Hasil aktivitas tadi akan dikirimkan kembali ke web browser untuk ditampilkan kepada pengguna.
