2. menu-menu perangkat lunak peta Minda​


Jawaban:

1.XMind. Xmind merupakan mind mapping software yang memiliki fitur lengkap. ...

2.FreeMind. Banyak para profesional yang menggunakan FreeMind sebagai software mind map mereka. ...

3.Edraw MindMap. Cara membuat mind map dengan Edraw MindMap tergolong sangat mudah. ...

4.MindMeister.

Penjelasan:

1. XMind

XMindXmind merupakan mind mapping software yang memiliki fitur lengkap. Software ini sangat mudah untuk digunakan. Walaupun XMind merupakan versi gratis dari XMind Pro, namun fiturnya terbilang paling lengkap dari software mind mapping gratis lainnya. Dengan XMind, anda bisa membuat Mind Map, Fishbone Diagram, serta spreadsheet. Dilengkapi dengan fitur markers, inline notes, serta hyperlink/attachments..XMind sangat mempermudah anda dalam melakukan brainstorming.

2. FreeMind

FreeMindBanyak para profesional yang menggunakan FreeMind sebagai software mind map mereka. Hal ini tidaklah mengherankan karena fungsi dan fitur dari FreeMind sangatlah lengkap. Namun demikian, memang secara tampilan software ini masih lebih sederhana jika dibandingkan software mind map gratis lainnya.

3. Edraw MindMap

Edraw MindMapCara membuat mind map dengan Edraw MindMap tergolong sangat mudah. Hal ini karena Edraw MindMap merupakan software pembuat mind map instan yang dilengkapi dengan fitur Smart Drawing Guide. Dilengkapi dengan banyak sekali template, style, efek dan banyak contoh mind map dalam berbagai topik, membuat Edraw MindMap menjadi software Mind Mapping yang cocok bagi pemula.

4. MindMeister

MindMeisterSebenarnya MindMeister bukanlah sebuah software, tetapi merupakan sebuah web aplikasi yang menyediakan layanan Mind Mapping online. Dengan mendaftarkan diri anda secara gratis, anda bisa menikmati fitur import, collaborate, share serta layanan MindMeister hingga 3 mind map.

12. menu dan icon perangkat lunak pengolah kata ( microsoft word)​


Jawaban:

1. File

2. Edit

3. View

4. Insert

5. Forma

6. Tools

7. Table

8. Windows

9. Help

Penjelasan:

1. File

Menu file berfungsi untuk membuat, membuka, menyimpan, mencetak, mengatur ukuran halaman hingga menutup suatu dokumen yang kita buat.

2. Edit

Menu edit berfungsi untuk mengedit suatu dokumen.

3. View

Menu view berfungsi untuk mengatur tampilan layar dari dokumen yang kita kerjakan.

4. Insert

Menu insert berfungsi untuk menyisipkan teks, tanggal atau waktu, symbol, diagram, file, onjek dan sebagainya.

5. Format

Menu format berfungsi untuk menetukan jenis huruf, paragraph, memberikan penomoran, membuat jenis kolom dalam teks, memberi bingkai dan arsiran, mengatur huruf besar dan kecil dan berbagai hal yang berkaitan dengan format dokumen.

6. Tools

Menu tools berfungsi untuk memeriksa ejaan, memilih bahasa yang dikehendaki.

7. Table

Menu table berfungsi untuk membuat table, menyisipkan table dalam teks dokumen, mengurutkan data, dan segala hal yang berkaitan table.

8. Windows

Menu windows berfungsi untuk membuka jendela baru, menyusun semua jendela yang aktif, dan melihat dokumen yang sedang aktif.

9. Help

Salah satu fungsi menu help adalah memunculkan fasilitas help yang bisa membantu kita menyelesaikan masalah.

19. Apa yang kamu ketahui sebelum mempelajari materi menu dan icon perangkat lunak pengolah kataSetelah mempelajari materi menu dan icon perangkat lunak pengolah kata apa yang dapat kamu pahami​


Jawaban:

Jawaban:

Manfaat perangkat lunak pengolah kata:

1. Membuat dan mengedit dokumen seperti surat dan laporan

Fungsi utama dari perangkat lunak pengolah kata, seperti Microsoft Word atau StarOffice adalah untuk membuat dan mengedit dokumen. Dengan perangkat lunak pengolah kata ini, dokumen dapat dibuat dengan lebih mudah karena ada banyak fitur di perangkat lunak ini, seperti fitur pengaturan ukuran kertas dokumen, font yang digunakan, pembuatan tabel dan sebagainya.

2. Membuat desain amplop

Selain dokumen, pengolah kata juga dapat membuat perlengkapan kantor lain seperti amplop. Dengan pengolah kata, kita bisa membuat nama dan tujuan dari amplop ini serta mengatur ukurannya.

3. Menyusun secara otomatis daftar isi dan tabel referensi

Dalam pembuatan berkas yang kompleks, seperti draft buku dan jurnal kita perlu membuat daftar isi hingga tabel dari referensi. Daripada harus menulis satu persatu dan mencocokkan nomor halaman dari bagian bab atau referensi ini, kita bisa menggunakan fitur-fitur di

4. Mencetak dokumen yang telah dibuat tersebut

Setelah dokumen yang kita buat selesai disusun, kita bisa mecetak dokumen ini. Dalam mencetak, perangkat pengolah kata, kita bisa mengatur ukuran kertas sesuai dengan yang ada printer. Kita juga bisa mengatur apakah pencetakan dilakukan bolak balik atau pada halaman yang lain.

5. Merubah dokumen yang dibuat sebagai format PDF

Pada beberapa versi terbaru pengolah kata, kita bisa mengubah file dokumen menjadi Portable Document Format (PDF). Format ini memudahkan dalam penyebaran, karena PDF mudah dipuka tanpa harus memiliki perangakt tertentu, serta menyimpan susunan dan font dokumen sehingga tampilannnya seragam pada perbagai sistem komputer.
