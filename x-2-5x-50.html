Pembahasan

[tex]\begin{aligned}&\underbrace{\sqrt[3]{\prod_{x=2}^4\left(\frac{x^2+2x}{x^2-1}\right)}}_{\begin{array}{c}a\end{array}}={}^n\log\underbrace{\left(\sum _{x=2}^5x^2+5x-6\right)}_{\begin{array}{c}b\end{array}}\\&\Large\text{$\Rightarrow {n}^a=b\quad....(i)$}\\&\qquad{\sf dengan\ }n > 0\,,\ n\ne1\end{aligned}[/tex]

 

Menentukan nilai a:

[tex]\begin{aligned}a&=\sqrt[3]{\prod_{x=2}^4\left(\frac{x^2+2x}{x^2-1}\right)}\\&=\sqrt[3]{\left(\frac{2^2+2(2)}{2^2-1}\right)\left(\frac{3^2+2(3)}{3^2-1}\right)\left(\frac{4^2+2(4)}{4^2-1}\right)}\\&=\sqrt[3]{\left(\frac{\cancel{8}}{3}\right)\left(\frac{\cancel{15}}{\cancel{8}}\right)\left(\frac{24}{\cancel{15}}\right)}\\&=\sqrt[3]{\frac{24}{3}}\ =\ \sqrt[3]{8}\\a&=\bf2\end{aligned}[/tex]

 

Menentukan nilai b:

[tex]\begin{aligned}b&=\sum_{x=2}^{5}\left(x^2+5x-6\right)\\&=\sum_{x=2}^{5}x^2\:+\:5\cdot\sum_{x=2}^{5}x\:-\:\sum_{x=2}^{5}6\\&=2^2+3^2+4^2+5^2\\&\quad\ +5(2+3+4+5)\\&\quad\ -(5-2+1)(6)\\&=4+9+16+25\\&\quad\ +5(14)\\&\quad\ -24\\&=54+70-24\\b&=\bf100\end{aligned}[/tex]

 

Menentukan nilai n dengan substitusi nilai a dan b ke pers. (i).

[tex]\begin{aligned}&\large\text{${n}^a=b$}\\&{\sf dengan\ }a=2\,,\ b=100\,,\ n > 0\,,\ n\ne1\\&\Rightarrow \large\text{${n}^2=100$}\\&\therefore\ \boxed{\ \large\text{$n=\bf10$}\ }\end{aligned}[/tex]

18. Quiz (+50): Diketahui → f(x) = 2x+4 → g(x) = 5x² buktikanlah jika (((fog)⁻¹)'')⁻¹(√2) = (¹/₂₀∛25)+4

1. Tentukan invers f(x) dan g(x)

[tex]\large\text{$\begin{aligned}&f(x) = 2x+4\\&\implies f^{-1}(x)=\frac{x-4}{2}\\\\&g(x) = 5x^2\\&\implies g^{-1}(x)=\pm\sqrt{\frac{x}{5}}=\pm\frac{1}{\sqrt{5}}\sqrt{x}\\\end{aligned}$}[/tex]

2. Tentukan (fog)⁻¹

[tex]\large\text{$\begin{aligned}(f\circ g)^{-1}(x)&=\left (g^{-1}\circ f^{-1}\right)(x)\\&=\Biggl.\pm\frac{1}{\sqrt{5}}\sqrt{\frac{x-4}{2}}\\&=\Biggl.\pm\frac{1}{\sqrt{10}}\sqrt{x-4}\end{aligned}$}[/tex]

3. Tentukan ((fog)⁻¹)'

[tex]\large\text{$\begin{aligned}\left((f\circ g)^{-1}\right)'(x)&=\Biggl.\pm\frac{1}{\sqrt{10}}\left(\frac{1}{2\sqrt{x-4}}\cdot(x-4)'\right)\\&=\Biggl.\pm\frac{1}{2\sqrt{10}\sqrt{x-4}}\end{aligned}$}[/tex]

3. Tentukan ((fog)⁻¹)''

[tex]\large\text{$\begin{aligned}\left((f\circ g)^{-1}\right)''(x)&\Biggl.=\pm\frac{1}{2\sqrt{10}}\left (\frac{1}{2\sqrt{(x-4)^3}}\cdot(x-4)' \right )\\&=\Biggl.\pm\frac{1}{4\sqrt{10}\sqrt{(x-4)^3}}\end{aligned}$}[/tex]

4. Tentukan (((fog)⁻¹)'')⁻¹

[tex]\large\text{$\begin{aligned}&\Biggl.y=\mp\frac{1}{4\sqrt{10}\sqrt{(x-4)^3}}\\{\iff}&\Biggl.y\cdot4\sqrt{10}\sqrt{(x-4)^3}=\mp1\\{\iff}&\Biggl.\sqrt{(x-4)^3}=\mp\frac{1}{4\sqrt{10}\cdot y}\\.....\ &\sf kuadratkan,\ tanda\ \mp\ hilang\\{\iff}&\Biggl.(x-4)^3=\frac{1}{160y^2}\\{\iff}&\Biggl.x-4=\frac{1}{\sqrt[3]{160y^2}}=\frac{1}{\sqrt[3]{8\cdot20y^2}}\\{\iff}&\Biggl.x-4=\frac{1}{2\sqrt[3]{20}\sqrt[3]{y^2}}=\frac{1}{2\cdot20^{\frac{1}{3}}\cdot\sqrt[3]{y^2}}\end{aligned}$}[/tex]

[tex]\large\text{$\begin{aligned}{\iff}&\Biggl.x-4=\frac{1\cdot20^{\frac{2}{3}}}{2\cdot20^{\frac{1}{3}}\cdot\sqrt[3]{y^2}\cdot20^{\frac{2}{3}}}\\{\iff}&\Biggl.x-4=\frac{20^{\frac{2}{3}}}{2\cdot20\sqrt[3]{y^2}}=\frac{\sqrt[3]{400}}{2\cdot20\sqrt[3]{y^2}}\\{\iff}&\Biggl.x-4=\frac{\sqrt[3]{8\cdot50}}{2\cdot20\sqrt[3]{y^2}}=\frac{\cancel{2}\sqrt[3]{50}}{\cancel{2}\cdot20\sqrt[3]{y^2}}\\{\iff}&\Biggl.x-4=\frac{1}{20}\sqrt[3]{\frac{50}{y^2}}\\{\iff}&\Biggl.x=\frac{1}{20}\sqrt[3]{\frac{50}{y^2}}+4\end{aligned}$}[/tex]

Sehingga, bentuk akhirnya adalah:
