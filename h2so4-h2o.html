19. setarakan NaOH+H2So4---->Na2So4+H2O


Cara penyetaraan reaksi

(1). aNaOH(aq) + bH₂SO₄(aq) → cNa₂SO₄(aq) + dH₂O(l)

Koefisien ruas kiri = koefisien ruas kanan
Na ⇒ a = 2c
O ⇒ a + 4b = 4c + d
H ⇒ a + 2b = 2d
S ⇒ b = c
Jika a = 2, maka c = 1, dan b = 1, lalu substitusikan ke a + 4b = 4c + d
menjadi 2 + 4 = 4 + d, sehingga diperoleh d = 2
Reaksi sudah setara:
2NaOH(aq) + H₂SO₄(aq) → Na₂SO₄(aq) + 2H₂O(l)
