8. jika periode f(x)= 2 cos(ax)=a adalah phi/3,maka nilai minimum fungsi f adalah


Jika periode f(x) = 2 cos (ax) + a adalah phi/3, maka nilai minimum fungsi f adalah 4

Bentuk fungsi trigonometri

y = A sin k (x ± b) ± c y = A cos k (x ± b) ± c y = A tan k (x ± b) ± c

dengan

A = amplitudo

Untuk fungsi trigonometri pada sinus dan kosinus

Besar periodenya: p = [tex]\frac{360^{o}}{k}[/tex]

Untuk fungsi trigonometri pada tangen

Besar periodenya: p = [tex]\frac{180^{o}}{k}[/tex]

Pembahasan

f(x) = 2 cos (ax) + a

Amplitudo ⇒ A = 2 Periode ⇒ p = [tex]\frac{\pi}{3}[/tex] k = a

p = [tex]\frac{\pi}{3}[/tex]

[tex] \frac{360^{o}}{k} = \frac{\pi}{3}[/tex]

[tex] \frac{2 \pi}{k} = \frac{\pi}{3}[/tex]

[tex] \frac{2}{k} = \frac{1}{3}[/tex]

k = 2 (3)

k = 6

Jadi fungsi trigonometrinya adalah

f(x) = 2 cos 6x + 6

Kita tahu bahwa nilai cos ax adalah terletak pada interval –1 ≤ cos ax ≤ 1

artinya  

nilai maksimum cos ax = 1 nilai minimum cos ax = –1

Jadi nilai minimum dari f(x) diperoleh jika cos 6x = –1, yaitu

f(x) = 2 cos 6x + 6

f(x) = 2 (–1) + 6

f(x) = –2 + 6

f(x) = 4

21. f(x) = 2 cos (ax) + a adalah π per 3. maka nilai minimum fungsi f adalah? mohon bantuannya:)


f(x) = a cos (kx + b) → periode : 2π/k atau 360°/k

fungsi f(x) = 2 cos (ax) + a
• a = 2
• k = a
• b = 0
periode 2π/k = 2π/a

diketahui di soal bahwa periodenya adalah π/3 maka :
π/3 = 2π/a
1/3 = 2/a
a = 6

→ f(x) = 2 cos (6x) + 6

rumus nilai maksimum dan minimum untuk bentuk fungsi :
f(x) = a sin (kx) + c atau
f(x) = a cos (kx) + c , adalah :

• nilai maksimum = (a) + c
• nilai minimum = -(a) + c

→ f(x) = 2 cos (6x) + 6
nilai minimum = -(a) + c
nilai minimum = -(2) + 6 = 4
