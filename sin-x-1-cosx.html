1. sin x/1+cosx - sin x/1-cosx​


1 + cos(x) = 2cos^2(x/2)

1 - cos(x) = -2sin^2(x/2)

sin(x) / (1 + cos(x)) - sin(x) / (1 - cos(x))

= sin(x) / (2cos^2(x/2)) - sin(x) / (-2sin^2(x/2))

= sin(x) / (2cos^2(x/2)) + sin(x) / (2sin^2(x/2))

2cos^2(x/2)sin^2(x/2).

= [sin(x)sin^2(x/2) + sin(x)cos^2(x/2)] / [2cos^2(x/2)sin^2(x/2)]

= [sin(x)[sin^2(x/2) + cos^2(x/2)]] / [2cos^2(x/2)sin^2(x/2)]

= sin(x) / [2cos^2(x/2)sin^2(x/2)]

Jadi, hasilnya adalah
(x) / [2cos^2(x/2)sin^2(x/2)].

25. lim x->1/2phi sin(cosx)/cosx


Jawaban:

Untuk mencari batas dari ekspresi tersebut, kita dapat menggunakan sifat-sifat trigonometri dan sifat limit.

Mari kita tulis ulang ekspresi tersebut:

lim x -> 1/2phi sin(cosx)/cosx

Kita tahu bahwa sin(x)/x memiliki batas saat x mendekati 0, yaitu 1.

Dan kita juga tahu bahwa cos(x) memiliki batas saat x mendekati 0, yaitu 1.

Kita dapat menggunakan kedua sifat tersebut untuk mencari batas dari ekspresi asli.

Jika kita menggunakan substitusi:

u = cos(x) saat x mendekati 0.

Maka batas x mendekati 0 sama dengan batas u mendekati 1 (karena u = cos(x), jadi saat x mendekati 0, u mendekati 1).

Menggunakan substitusi tersebut, kita dapat menulis ulang ekspresi awal:

lim u -> 1 sin(u)/u

Kita tahu bahwa batas dari sin(u)/u saat u mendekati 0 adalah 1.

Sehingga kita dapat menyimpulkan bahwa batas ekspresi asli saat x mendekati 1/2phi adalah 1.
