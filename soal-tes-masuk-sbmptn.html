9. SOAL UJIAN MASUK UM UGM 2013 SBMPTN 2016


Jawaban: C

Soal ini lebih mudah diselesaikan dengan mensubstitusikan pilihan jawaban ke soal dan mengeliminasi jawaban yang salah:

Ambil x = 2, maka diperoleh:

3√(2 - 2)/ 2 - 1 < 2 ⇒ 0 < 2 (BENAR)

Pilihan jawaban yang tidak memuat x = 2 salah sehingga pilihan jawaban A, B, E salah.

Kemudian ambil x = 0, maka diperoleh:

3√(2 - 0)/ 0 - 1 < 2 ⇒ 3√2/-1 < 2 (BENAR)

Pilihan jawaban yang tidak memuat x = 0 salah sehingga pilihan jawaban D salah.
Jadi, jawaban yang benar adalah C.

22. syarat masuk sbmptn?


Persyaratan SBMPTN 2017 untuk sekolah :
1. SMA/SMK/MA negeri maupun swasta, termasuk sekolah RI di luar negeri yang mempunyai NPSN.
2. Telah mengisi PDSS dengan lengkap dan benar.

Persyaratan SBMPTN 2017 untuk siswa :
1. Calon peserta adalah siswa SMA/MA, SMK kelas terakhir pada tahun 2017
2. Memiliki prestasi unggul yaitu : calon peserta masuk peringkat terbaik di sekolah pada semester tiga, empat dan lima, dengan ketentuan berdasarkan akreditasi sekolah.
3. Siswa memiliki NISN dan terdaftar pada PDSS.
4. Siswa memiliki nilai rapot semester 1 sampai semester 5 yang telah diisikan pada PDSS.
5. memenuhi persyaratan lain yang ditentukan oleh masing2 PTNlulus seleksi

29. SOAL UJIAN MASUK UM UGM 2013 SBMPTN 2016


Jawaban: B

Karena (b + c, b, c) memenuhi persamaan, maka:
3(b + c) - b + 2c = -1
3b + 3c - b + 2c = -1
2b + 5c = -1........................Pers.1

-2(b + c) + b + 3c = -3
-2b - 2c + b + 3c = -3
-b + c = -3...........................Pers.2

2b + 5c = -1
-2b + 2c = -6
_____________ +

7c = -7
c = -1

-b + (-1) = -3
b = 2

Jadi, b + c = 2 + (-1) = 1
