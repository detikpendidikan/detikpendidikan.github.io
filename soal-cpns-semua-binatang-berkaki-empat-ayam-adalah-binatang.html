5. Negasi dari pernyataan “ Semua binatang berkaki empat” adalah


Negasi dari "Semua binatang berkaki empat" adalah tidak semua binatang berkaki empat atau ada binatang tidak berkaki empat. Pernyataan pada soal termasuk pernyataan dengan kuantor universal. Negasi kuantor universal dapat menggunakan kuantor sebagian. Simak penjelasan berikut.

Penjelasan dengan langkah-langkah

Materi

Logika Matematika

Berikut istilah-istilah, lambang, dan kata hubung dalam pernyataan logika matematika.

Negasi ( ~ ) → tidak, bukanKonjungsi ( ∧ ) → dan, tetapi, walaupun, kemudianDisjungsi ( ∨ ) → atauImplikasi ( ⇒ ) → jika …… maka ……Biimplikasi ( ⇔ ) → jika dan hanya jika …… maka ……Kuantor Universal ( ∀(x) ) → semua …..Kuantor Sebagian ( ∃(x) ) → ada, sebagian

Ingkaran/Negasi (~)

Kuantor Universal → ~ [∀(x), p(x)] ≡ ∃(x), ~ p(x)Kuantor Sebagian → ~ [∃(x), p(x)] ≡ ∀(x), ~ p(x)Konjungsi (p ∧ q) → ~ (p ∧ q) ≡ ~ p ∨ ~ qDisjungsi (p ∨ q) → ~ (p v q) ≡ ~ p ∧ ~ qImplikasi (p ⇒ q) → ~ (p ⇒ q) ≡  p ∧ ~ qBiimplikasi (p ⇔ q) → ~ (p ⇔ q) ≡ (p ∧ ~ q) ∨ (q ∧ ~p)

Diketahui

Pernyataan: "Semua binatang berkaki empat".

Ditanya

Apa negasi dari pernyataan tersebut?

Jawab

Semua binatang berkaki empat termasuk kuantor universal ∀(x). Oleh karena itu, negasi dari pernyataan tersebut dapat ditentukan dengan langkah berikut.

∀(x), p(x) = Semua binatang berkaki empat.

∀(x) = Semua binatang.

p(x) = berkaki empat.

Negasinya:

~ [∀(x), p(x)] ≡ ∃(x), ~ p(x)

Tidak semua binatang berkaki empat

atau

Ada binatang tidak berkaki empat.

Kesimpulan

Jadi, negasi dari "Semua binatang berkaki empat" adalah tidak semua binatang berkaki empat atau ada binatang tidak berkaki empat.

24. binatang peliharaan pak made 37 ekor terdiri dari ayam dan babi jika jumlah seluruh kaki adalah 88 tentukan berapa masing masing binatang itu ? jelaskan


A = ayam, 2 kaki
B = babi, 4 kaki

A + B = 37
A = 37 - B

2A + 4B = 88
2 (37 - B) + 4B = 88
74 - 2B + 4B = 88
2B = 88 - 74
2B = 14
B = 14/2 = 7 (jumlah babi)

A = 37 - B
A = 37 - 7 = 30 (jumlah ayam)
