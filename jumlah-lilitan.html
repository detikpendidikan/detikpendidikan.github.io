dit Vs...?

[tex] \frac{np}{ns} \times \frac{vp}{vs} \\ 200vs = 220 \times 600 \\ vs= \frac{220 \times 600}{200} \\ vs = 220 \times 3 \\ = 660[/tex]

Vs = (Ns x Vp)/Np

Vs = (600 x 220)/200

Vs = 660 volt

Jawaban:

Penjelasan:

Ciri ciri Transformator Step Up :

Vs > VpNs > NpIp > Is

Ciri Ciri Transformator Step Down :

Vp > VsNp > NsIs > Ip

Keterangan

Vp : Tegangan Primer

Np : Lilitan Primer

Ns : Lilitan Sekunder

Ip : Arus Primer

Is : Arus Sekunder

25. apa yang kamu ketahui tentang transformator dengan jumlah lilitan pada kumparan sekunder lebih banyak daripada jumlah lilitan pada kumparan primer


Jika Ns > Np , dari hubungan pada trafo bahwa

Vp : Vs = Np : Ns

maka

Vs > Vp tegangan sekunder lebih besar dari tegangan primer.

Trafo jenis ini disebut trafo STEP UP karena menaikkan tegangan.

Selain itu juga berlaku hubungan arus dgn jumlah lilitan.

Ip : Is = Ns : Np

Karena Ns > Np maka arus primer lebih besar dari arus sekunder.

Jadi dengan Ns > Np maka

Vs > Vp

dan

Is < Ip

Untuk trafo ideal, daya input sama dgn daya output.

Vp. Ip = Vs. Is

Transformator

------------------------

Transformator step up berfungsi untuk menaikkan tegangan

Transformator step down berfungsi untuk menurunkan tegangan

keterangan rumus :

Is = kuat arus sekunder

Ip = Kuat arus primer

Vp = tegangan primer

Vs =tegangan sekunder

Np = lilitan primer

Ns = lilitan sekunder

Jika sebuah transformator bertipe step up maka

-.Vs lebih besar dari Vp

-. Ns lebih besar dari Np

Jika sebuah transformator bertipe step down maka

-. Vp lebih besar dari Vs

-. Np lebih besar dari Ns

Tambahan rumus :

Perhitungan efesiensi trafo (π)

π = Pout /Pin x 100%

Keterangan rumus :

Pout = daya listrik pada kumparan sekunder

Pin = daya listrik pada kumparan primer
