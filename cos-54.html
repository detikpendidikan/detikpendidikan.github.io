= ½(sin (54 + 36) - sin (54 - 36))

= ½(sin 90 - sin 18)

= ½(1 - sin 18)

= ½ - ½ sin 18

• sin 54 . cos 36

= ½(2 sin 54 . cos 36)

= ½(sin (54 + 36) + sin (54 - 36))

= ½(sin 90 + sin 18)

= ½(1 + sin 18)

= ½ + ½ sin 18

= cos 54 . sin 36 + sin 54 . cos 36

= (½ - ½ sin 18) + (½ + ½ sin 18)

= ½ + ½ - ½ sin 18 + ½ sin 18

11. Sin 18 Cos 18 Cos 54 - Sin 54 Sin²18 + 2 Cos 18 Sin 36 Cos 36​


Mari kita pecah dan hitung ekspresi trigonometri ini langkah demi langkah:

Sin(18°) = Sin(36° / 2) = √[(1 - Cos(36°)) / 2]

Cos(18°) = Cos(36° / 2) = √[(1 + Cos(36°)) / 2]

Cos(54°) = Cos(36° * 3/2) = -√[(1 + Cos(36°)) / 2]

Sin(54°) = Sin(36° * 3/2) = √[(1 - Cos(36°)) / 2]

Sekarang kita bisa menggantikan nilai-nilai ini ke dalam ekspresi awal:

Sin(18°) * Cos(18°) * Cos(54°) - Sin(54°) * Sin²(18°) + 2 * Cos(18°) * Sin(36°) * Cos(36°)

= [√(1 - Cos(36°)) / 2] * [√(1 + Cos(36°)) / 2] * [-√(1 + Cos(36°)) / 2] - [√(1 - Cos(36°)) / 2] * [√(1 - Cos(36°)) / 2] + 2 * [√(1 + Cos(36°)) / 2] * [√(1 - Cos(36°)) / 2]

Sekarang kita bisa menyederhanakan lebih lanjut:

= -[1 - Cos(36°)]/4 - [1 - Cos(36°)]/4 + √[(1 + Cos(36°))(1 - Cos(36°))]

= -[1 - Cos(36°)]/2 + √(1 - Cos²(36°))

Kemudian, kita tahu bahwa Cos(36°) = (1 + √5) / 4 (ini adalah nilai Cosinus sudut 36 derajat yang dapat ditemukan dari rasio emas atau Golden Ratio). Maka:

= -[1 - (1 + √5) / 4]/2 + √[1 - ((1 + √5) / 4)²]

= -[3 - √5]/8 + √[1 - (1 + √5)² / 16]

= -[3 - √5]/8 + √[16/16 - (1 + √5)² / 16]

= -[3 - √5]/8 + √[15 - 2√5] / 4

= (-3 + √5)/8 + √(15 - 2√5) / 4

Jadi, hasil akhir dari ekspresi ini adalah:

(√5 - 3)/8 + √(15 - 2√5)/4

[tex]\begin{aligned}\cos 36^\circ&=1-2\sin^2 18^\circ\\&=1-2\left ( \frac{\sqrt{5}-1}{4} \right )^2\\&=1-\frac{3-\sqrt{5}}{4}\\&=\frac{1+\sqrt{5}}{4}\end{aligned}[/tex]

[tex]\begin{aligned}\sin 36^\circ&=\sqrt{1-\cos^2 36^\circ}\\&=\sqrt{1-\left ( \frac{1+\sqrt{5}}{4} \right )^2}\\&=\sqrt{1-\frac{6+2\sqrt{5}}{16}}\\&=\sqrt{\frac{10-2\sqrt{5}}{16}}\\&=\sqrt{\frac{5-\sqrt{5}}{8}}\\&=\frac{\sqrt{5-\sqrt{5}}}{2\sqrt{2}}\end{aligned}[/tex]

Cari cos 18° berdasarkan rumus cos² x = 1 - sin² x

[tex]\begin{aligned}\cos 18^\circ&=\sqrt{1-\sin^2 18^\circ}\\&=\sqrt{1-\left ( \frac{\sqrt{5}-1}{4} \right )^2}\\&=\sqrt{1-\frac{6-2\sqrt{5}}{16}}\\&=\sqrt{\frac{10+2\sqrt{5}}{16}}\\&=\sqrt{\frac{5+\sqrt{5}}{8}}\\&=\frac{\sqrt{5+\sqrt{5}}}{2\sqrt{2}}\end{aligned}[/tex]

Maka

[tex]\begin{aligned}&\frac{\cos 54^\circ}{\cos 18^\circ}\sin 36^\circ\\&=\frac{\sqrt{\frac{5-\sqrt{5}}{8}}}{\sqrt{\frac{5+\sqrt{5}}{8}}}\sqrt{\frac{5-\sqrt{5}}{8}}\\&=\sqrt{\frac{5-\sqrt{5}}{5+\sqrt{5}}}\sqrt{\frac{5-\sqrt{5}}{8}}\\&=\sqrt{\frac{5-\sqrt{5}}{5+\sqrt{5}}~\frac{5-\sqrt{5}}{8}}\\&=\sqrt{\frac{30-10\sqrt{5}}{40+8\sqrt{5}}}\\&=\sqrt{\frac{5\left ( 6-2\sqrt{5} \right )}{4\left ( 10+2\sqrt{5} \right )}}\end{aligned}[/tex]

23. #sertakan_cara Nilai dari sin 54° + cos 108° ?


Dengan menggunakan identitas trigonometri

sin A = cos(90 - A), maka

sin 54° = cos 36°


maka bentuk pada soal diubah menjadi

cos 36° + cos 108°

= 2 cos ½(36+108) cos ½(36-108)

= 2 cos 72° cos 36°


Buat pemisalan


cos 36 - cos 72 = x

maka

x² = cos² 36 - 2 cos 36 cos 72 + cos² 72

x² = ½(1 + cos 72) - 2 cos 36 cos 72 + ½(1 + cos 144)

x² = ½(2 + cos 72 - cos 36) - 2 cos 36 cos 72

x² = 1 - ½x - (cos 108 + cos 36)

x² = 1 - ½x - (-cos 72 + cos 36)

x² = 1 - ½x - x

x² = 1 - 3/2 x

2x² + 3x - 2 = 0

(2x - 1)(x + 2) = 0

x = ½


Maka

cos 36 - cos 72 = x

cos 36° + cos 108° = x

2 cos 72° cos 36° = x = ½

cos 72° cos 36° = ¼


Maka dari soal di atas

2 cos 72° cos 36° = 2 (¼) = ½ ✔️
