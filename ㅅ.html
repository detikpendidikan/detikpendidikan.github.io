22. tentukan persamaan garis lurus melalui titik A (0,1) dan B (3,4) kaka tolongg yaa(｡>ㅅ<｡)​


-Persamaan garis-

[tex] \frac{y - y1}{y2 -y1} = \frac{x - x1}{x2 - x1} [/tex]

[tex] \frac{y - 1}{4 - 1} = \frac{x - 0}{3 - 0} [/tex]

[tex] \frac{y - 1}{3} = \frac{x - 0}{3} [/tex]

[tex]3(y - 1) = 3(x - 0)[/tex]

[tex]3y - 3 = 3x[/tex]

[tex]3y = 3x + 3[/tex]

[tex]y = \frac{3x + 3}{3} [/tex]

[tex]y = x + 1[/tex]
