2. Tolong dibantu soal MTK poltekes


5 = p² - 2(-3p+1) 
5 = p² + 6p - 2 
p² + 6p - 7 = 0 
(p+7)(p-1) 
berarti nilai p = -7 atau p = 1 
math is easyMapel : Matematika
Kelas : X SMA
Bab : Persamaan Kuadrat

Pembahasan :
x² + px - 3p + 1 = 0

Berdasarkan Teorema Vietta
X1 + X2 = -b/a
X1 + X2 = -p

X1 . X2 = -3p + 1

Maka...
X1² + X2² = 5
(X1 + X2)² - 2(X1 . X2) = 5
(-p)² - 2(-3p + 1) = 5
P² + 6p - 2 = 5
P² + 6p - 7 = 0
(P + 7)(p - 1) = 0
P = 1
P = -7
