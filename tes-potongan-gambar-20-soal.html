23. Soal Tes Diketahui fungsi Y = 16 – X2, jika akan menggambarkan kurvanya, carilah : 1. titik potong dengan sumbu absis (X) 2. titik potong dengan sumbu ordinat (Y) 3. berapa besarnya Y jika X = 2 ? 4. jika fungsi tersebut akan digambarkan dalam diagram kartesius, bagaimana kurvanya ?


Jawaban:

Penjelasan:

Fungsi y = 16 - x²

Titik potong dengan sumbu x ---> y = 0

y = 16 - x²

0 = 16 - x²

x² = 16

x1 = 4 dan x2 = -4

Titik potong nya adalah ( 4 , 0 ) dan ( -4 , 0 )

Titik potong sumbu y ----> x = 0

y = 16 - x²

y = 16 - 0²

y = 16

Titik potong nya ( 0 , 16 )

Nilai y jika x = 2

y = 16 - x²

y = 16 - 2²

y = 16 - 4

y = 12

Lihat gambar yang dilampirkan. Kurva terbuka ke bawah.
