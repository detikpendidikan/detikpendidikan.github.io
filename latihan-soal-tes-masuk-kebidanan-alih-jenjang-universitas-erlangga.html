cos²t = (1/a + 1/b)²

sin²t + cos²t = (1/a² + 1/b² - 2/ab) +  (1/a² + 1/b² + 2/ab)

25. Soal latihan UM UGM Ujian Masuk Universitas Gadjah Mada


U5/U10 = (a + 4b)/(a + 9b)

4(U10/U5) = 4((a + 9b)/(a + 4b))
= (4a + 36b)/(a + 4b)

S5/S10 = 2/3
(a + a + b + a + 2b + a + 3b + a + 4b)/(a + a + b + a + 2b + ... + a + 9b) = 2/3

(5a + 10b)/(10a + 45b) = 2/3
3(5a + 10b) = 2(10a + 45b)
15a + 30b = 20a + 90b
20a - 15a = 30b - 90b
5a = -60b
a = -12b

U5/U10 = (a + 4b)/(a + 9b)
= (-12b + 4b)/(-12b + 9b)
= -8b/-3b
= 8/3

4(U10/U5) = (4a + 36b)/(a + 4b)
= (4(-12b) + 36b)/(-12b + 4b)
= (-48b + 36b) / (-8b)
= (-12b)/(-8b)
= 3/2

log(U5/U10 - 4(U10/U5)) = log(8/3 - 3/2)
= log((16 - 9)/6)
= log(7/6)
= log7 - log6
